# 📖 使用指南 - 文字与代码匹配可视化

## 🎯 功能概述

这个工具能够自动分析文本描述与代码片段的对应关系，并提供详细的可视化展示。你可以清楚地看到哪段文字与哪段代码相匹配，以及它们的相似度。

## 🚀 快速开始

### 1. 运行演示
```bash
# 查看基本功能演示
python demo.py

# 查看匹配可视化演示
python demo_matches.py

# 启动完整Web应用
streamlit run main.py
```

### 2. 基本工作流程

1. **📝 输入文本** - 上传论文、文档或粘贴文本内容
2. **💻 输入代码** - 连接GitHub仓库或上传代码文件
3. **🔗 运行对齐** - 执行智能对齐分析
4. **🔍 查看匹配** - 在"Match Details"页面查看详细匹配

## 🔍 匹配可视化功能

### 主要视图

#### 1. 详细匹配列表
- **功能**: 显示所有找到的文本-代码匹配
- **特点**: 
  - 按相似度排序
  - 可过滤和搜索
  - 显示匹配详情和解释

#### 2. 并排对比视图
- **功能**: 并排显示匹配的文本和代码
- **特点**:
  - 清晰的对比展示
  - 统计信息显示
  - 匹配分析

#### 3. 交互式仪表板
- **功能**: 综合的可视化分析
- **包含**:
  - 匹配列表
  - 统计分析
  - 网络图

### 匹配信息展示

每个匹配项包含：

#### 📝 文本信息
- **内容**: 匹配的文本片段
- **位置**: 在原文中的位置
- **长度**: 字符数和单词数
- **类型**: 句子、段落、公式等

#### 💻 代码信息
- **内容**: 匹配的代码片段
- **文件**: 代码文件路径
- **行号**: 在文件中的行号范围
- **语言**: 编程语言类型

#### 🔗 匹配详情
- **相似度**: 0-1之间的相似度分数
- **置信度**: 匹配的可靠性
- **类型**: 语义、词汇、结构匹配
- **解释**: 为什么这两个片段匹配

## 📊 可视化图表

### 1. 相似度分布图
- **用途**: 查看所有匹配的相似度分布
- **信息**: 高/中/低相似度匹配的数量

### 2. 匹配类型分布
- **用途**: 了解匹配的类型分布
- **类型**: 语义匹配、结构匹配等

### 3. 网络图
- **用途**: 可视化文本与代码的连接关系
- **特点**: 节点表示文本/代码，边表示匹配关系

## 🎛️ 交互功能

### 过滤器
- **相似度阈值**: 只显示高于指定阈值的匹配
- **匹配类型**: 按类型筛选匹配
- **显示数量**: 控制显示的匹配数量

### 排序选项
- **按相似度**: 从高到低排序
- **按置信度**: 按可靠性排序
- **按类型**: 按匹配类型分组

### 搜索功能
- **文本搜索**: 在文本内容中搜索
- **代码搜索**: 在代码内容中搜索
- **关键词**: 按关键词过滤

## 📈 使用技巧

### 1. 提高匹配质量
- **文本质量**: 使用清晰、结构化的文本
- **代码质量**: 确保代码有良好的注释和命名
- **相似度阈值**: 调整阈值以获得最佳结果

### 2. 理解匹配结果
- **高相似度 (>0.8)**: 强匹配，文本与代码高度相关
- **中等相似度 (0.6-0.8)**: 中等匹配，有一定相关性
- **低相似度 (<0.6)**: 弱匹配，相关性较低

### 3. 分析匹配模式
- **语义匹配**: 基于含义的匹配
- **结构匹配**: 基于代码结构的匹配
- **词汇匹配**: 基于关键词的匹配

## 🚨 注意事项

### 1. 数据质量
- 确保文本和代码的质量
- 避免过于简短或模糊的内容
- 保持文本和代码的相关性

### 2. 性能考虑
- 大文件可能需要较长处理时间
- 建议分批处理大量数据
- 注意内存使用情况

### 3. 结果解释
- 相似度不是绝对的准确性指标
- 需要人工验证重要的匹配
- 结合上下文理解匹配结果
